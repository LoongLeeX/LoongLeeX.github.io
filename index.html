Welcome to W3.CSS Basic exampleSite

  • Built with W3.CSS
  • Highly configurable
  • Change all colors in the blink of an eye

W3.CSS Basic Theme

  • More than 100 predefined color themes
  • More than 25 shortcodes
  • Support for image processing and attachments

Responsive design

  • Mobile first
  • 3 (medium, lage) or 1 (small) column layout
  • Small menu (first characters) on small screen (no collapse)

Customizable landing page

  • Optional Marquee, Jumbotron
  • Optional Photocards, Features
  • Optional Recent Posts, See More
  • Optional Clients, Testimonials

Optional Google Features

  • Fonts - either via api or via local files
  • Translate Widget
  • Global Site Tags with your GA_TRACKING_ID
  • Map on Contact page
  • Search page with google customizable search api

Cool stuff included

  • Fontawesome version 5
  • Owl Carousel
  • Animate.css
  • PhotoSwipe based Gallery with image processing (resource)

Contact page

  • Optional formspree.io
  • You may configure mail feedback link (privacy)

Blog

  • Optional Disqus
  • Optional privacy protection (disable count) for Disqus
  • Lists for taxonomy and termy

Quite unique

  • Alternative navigation (Top/Up/Down & pages on this level)
  • Optional fine grained tracing via embedded comments
  • Shorten menu entries on small displays
  • Custom Summary using truncate

If privacy matters

  • Works with disabled javascript (using fallbacks)
  • Local fonts
  • Libs are local (no cdn)
  • Google stuff can be disabled

This theme is for

  • Landing pages
  • Blogs
  • Pages in a directory structure with simple navigation
  • People who like monochromatic colors
