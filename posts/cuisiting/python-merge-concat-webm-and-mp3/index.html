Install

  1. 安装ffmpeg
brew install ffmpeg
  1. 安装 ffmpeg 的python api
    注意是 pip install ffmpeg-python
    instead of pip install ffmpeg
pip install ffmpeg-python

如果已经安装了 module 为 ffmpeg 的api，则先uninstall
注意此处区分 brew install 和 pip install 的区别


pip uninstall ffmpeg
pip uninstall ffmpeg-python

pip install ffmpeg-python

Run

import ffmpeg  
  
  
video_path = ''  
audio_path = ''  
input_video = ffmpeg.input(video_path)  
input_audio = ffmpeg.input(audio_path)  
  
ffmpeg.concat(input_video, input_audio, v=1, a=1).output('complete.mp4').run()
