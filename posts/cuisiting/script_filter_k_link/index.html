在配置Alfred Script Filter 时,
若要打开的文件路径 为

/Users/your_user/Library/Application\ Support/xxxxxx

这类中间带有空格的路径, Alfred 将不能正常打开

{
		"uid": "kconfig",
		"type": "file",
		"title": "k config",
		"subtitle": "~/Desktop",
		"arg": ["vscode",
				"/Users/your_user/toolchain/MyShell/Alfred/script_filter_k_link/info.plist"],
		"autocomplete": "Desktop",
		"icon": {
			"type": "fileicon",
			"path": "~/Desktop"
		}
	},

解决办法:
为 上述文件创建文件链接

格式：ln [原文件] [硬链接文件]
